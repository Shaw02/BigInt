多倍長整数クラス for Microsoft Visual C++ (32bit)　メンバー関数

BigInt（コンストラクタ、デストラクタ）

定義

コンストラクタ

	BigInt::BigInt(void);

	BigInt::BigInt(BigInt&	_Src);

	BigInt::BigInt(int		_Src);

	BigInt::BigInt(char* _ptr, unsigned int _sz);

デストラクタ

	BigInt::~BigInt(void);

引数

BigInt& _Src … 設定する多倍長整数のオブジェクト
int _Src … 設定する整数値
char* _ptr … 設定する整数値のあるポインタ
unsigned int _sz … 設定する整数値のバイト長

戻り値

無し

内容

引数を設定しない場合は、整数値は０となります。
引数を設定する事で、整数値を初期化できます。

  • BigInt型のクラスオブジェクトによる設定
  • int型での設定
  • メモリ中の整数をポインタとバイト長で設定するコンストラクタがあります。
